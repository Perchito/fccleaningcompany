FC Cleaning Company static website

Files included:
- index.html
- services.html
- about.html
- contact.html
- styles.css
- FC-CLEANING-COMPANY-LOGO.jpg

How to use:
1. Keep all files in the same folder.
2. Open index.html in a browser to preview the site.
3. Replace placeholder phone/email details with your real details.
4. To publish, upload the whole folder to Netlify, Vercel, GitHub Pages, or standard web hosting.

Note:
The contact form is now set up for Netlify Forms. To make it work, deploy the site to Netlify so form detection can process the HTML form and collect submissions.


Netlify setup:
1. Create a free Netlify account.
2. Drag and drop the unzipped site folder into Netlify Deploys, or connect a GitHub repo.
3. After deployment, Netlify will detect the form because the form uses name="contact" and data-netlify="true".
4. Form submissions will appear in your Netlify dashboard under Forms.
5. You can set email notifications inside Netlify so each enquiry is emailed to you.
